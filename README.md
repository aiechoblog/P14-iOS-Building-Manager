# P14-iOS-Building-Manager


P14 is a Persian iOS/iPadOS building management app designed for small residential buildings.

The app helps a building manager track:

- Building units
- Owners and tenants
- Monthly charges
- Payments
- Expenses
- Recurring utility bills
- Opening fund balance
- Debtors
- Monthly reports
- Copyable resident messages

## Current Status

This project started as a Python prototype on iPad using Pyto and is now being developed as a SwiftUI iOS/iPadOS app.

## Main Goal

The goal is to build a simple, practical, single-user building management app that can run locally on iPhone and iPad.

## Platform

- iOS / iPadOS
- SwiftUI
- Swift Playgrounds compatible
- Local JSON storage
- Persian UI
- Right-to-left layout
